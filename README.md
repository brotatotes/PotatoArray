# PotatoArray

## Try it

`pip3 install PotatoArray`

#### In python3:

```
>>> from PotatoArray import PotatoArray
>>> a = PotatoArray([1,2,3])
>>> a[2]
1
>>> a[-2]
3
```


## Description

A wrapper class around python3's list with index starting at 2.
